

# PACK-D: Standardized Workpack Management System

PACK-D is a web-based workpack management system developed as a Final Year Project (FYP) to improve the preparation, submission, tracking, and approval of engineering workpacks through a centralized digital platform.

The system is designed to address inefficiencies in conventional manual workpack workflows by providing standardized documentation, approval tracking, risk & safety management, supporting document uploads, meeting  scheduling, and reporting features.


## Project Objective
The project aims to standardize engineering workpack processes and improve workflow transparency, traceability, and operational efficiency within project-based environments.

 # Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
